# AI Creative Engine

AI Creative Engine is a lightweight AI chatbot built using FastAPI and HuggingFace Transformers.  
It allows users to enter a prompt in a simple web interface and receive AI-generated responses.

## 🚀 Tech Stack

- Python
- FastAPI
- HuggingFace Transformers
- HTML / CSS
- JavaScript Fetch API

## ✨ Features

- Local AI text generation
- Simple chatbot interface
- FastAPI backend
- Clean minimal UI
- Runs locally without external APIs
